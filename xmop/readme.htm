XML Metadata Object Persistence (XMOP) Readme

Jonathan Borden mailto:jborden@mediaone.net

This version includes support for SOAP and XML-RPC calls from

IE5 javascript client -> IIS/ASP COM server

-> Apache/Tomcat 3.1 Java Servlets

Dependencies:

The java servlets have been tested under Tomcat 3.1

Obtain the Xalan (XSLT engine) and Xerces (XML parser) projects.

These projects are all obtainable at http://xml.apache.org

 

The COM/ASP project has been built with MSVC5 and uses the ObjectSpace STL classes (www.objectspace.com). It requires MSXML2 (the preview is currently available).
