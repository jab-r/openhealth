The Open Healthcare Group

A Smarter Way for Better Healthcare

The Open Healthcare Group is an organization devoted to the promotion and distribution of an open source health record, XChart. Using the latest XML Internet technology, we have developed the source code for a health record that will help you automate and streamline the practice of medicine.
